# Orders Microservice

## Dev

1. Clone rep
2. Install dependencies
3. Create `.env` file based on `env.example`
4. Exec prisma migration `npx prisma migrate dev`
5. ```
   docker compose up -d
   ```
6. Exec `npm run start:dev`
